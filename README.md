# Cyber Threat Intelligence Platform

## Overview
ML-powered Cyber Threat Intelligence system with threat correlation and MITRE ATT&CK mapping.

## Features
- Threat correlation engine
- Random Forest threat prioritization
- MITRE ATT&CK mapping
- Real-time alerts
- Interactive dashboard

## Tech Stack
Python
Flask
JavaScript
Chart.js

## Installation
git clone https://github.com/username/cti-platform
cd cti-platform
pip install -r requirements.txt
python app.py
